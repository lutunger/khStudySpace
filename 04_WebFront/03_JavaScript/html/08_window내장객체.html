window 객체

        - 브라우저 창 자체를 나타내는 객체

        - window 객체는 자바스크립트의 최상위 객체이며
            DOM, BOM으로 분류된다

        DOM(Document Object Model) : document

        BOM(Browser Object Model) : location, history, screen, navigator(브라우저 정보)

        + DOM, BOM 말고도 window가 자체적으로 가지고 있는 기능도 존재
        ex) alert(), confirm(), prompt() 등...

        -> window가 가지고 있는 함수들은
        창 자체에 종속되서 동작하기 때문에 
        window 객체를 생략하고 호출할 수 있다

window.setTimeout(함수, 지연시간(ms))

        지연시간이 지난 후 함수 수행되게 만드는 window 내장 함수

window.setInterval(함수, 지연시간(ms))

        지연시간이 지날 때마다 함수를 수행되게 만드는 window 내장 함수
